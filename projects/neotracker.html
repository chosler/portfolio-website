Neo-Tracker

NEO-Tracker was inspired by my fascination with space. The idea was to create an application that would allow users to see basic information about Near Earth Objects (NEOs), specifically asteroids, and be able to map those objects in a 3D model of the solar system. I also created a fully rendered 3D model of a single asteroid using data obtained by NASA.

Name: NEO-Tracker
GitHub: FrontEnd
BackEnd
Technologies: Ruby on Rails, HTML, CSS, Javascript, React, JWT, NASA Open APIs, Spacekit.js Library
