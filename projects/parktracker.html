ParkTracker

I've always been captivated by the National Park Service, whenever I get the chance I love going to the parks (I even collect pins from every Park I've been to). So the idea behind ParkTracker was to make a application that would allow users to keep a collection of the Parks that they've been to or would like to go to. There is also live weather data and an interactive map for each Park.

Name: ParkTracker
GitHub: FrontEnd
BackEnd
Technologies: Ruby on Rails, HTML, CSS, Javascript, React, National Park Service API, OpenWeatherMap API, MapBox Library
